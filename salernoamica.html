Description

Project developed for an exam and the App Challenge IX Edition. The application was created for the municipality of Salerno and consists of an aggregation point for all the services offered by the municipality of Salerno. The main functionalities are: service search, defibrillator insertion and search, and the ability to ask for help from the chatbot for any problem. The group was composed of 4 people. We used Flutter for front-end development, Spring for back-end development, MySQL for the database, OliaviaAI for the chatbot, and Firebase for notification management.

Technologies

  • Flutter: Front-end user friendly, focus on usability and accessibility
  • Spring: Back-end
  • Firebase: Send message to device
  • OpenStreetMap: Service Geolocation on Map
  • OliviaAI: Chatbot that can answer questions and can suggest a navigation path to the answer.
